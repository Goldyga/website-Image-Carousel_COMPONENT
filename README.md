# Images carousel

This is next simple website component based on Jquery. The gallery is moving to left and right and view another images.

[Live version](https://goldyga.github.io/website-Image-Carousel_COMPONENT/)

* HTLM
* SCSS
* GRUNT
* GRID
* JQUERY
